# 🍲 Recipe Book App
A comprehensive Android application for food enthusiasts to manage, share, and discover culinary inspirations with advanced productivity and social tools.

## 🚀 Key Features

- **User Authentication:** Complete login, registration, and "Forgot Password" flow using Firebase Auth.
- **Personal Recipe Manager:** Users can create, update, and delete their own recipes.
- **Image Integration:** Support for uploading and displaying recipe photos via Firebase Storage/Cloudinary.
- **Search & Filter:** Easily find recipes by name or ingredients.
- **User Profiles:** Customizable profiles including personal details and profile pictures.
- **Favorites System:** Toggle "Favorite" status to save recipes in a dedicated collection for quick offline-like access.
- **Social Engagement:** Real-time "Likes" and "Comments" on public recipes to foster a cooking community.
- **Digital Shopping List:** Automatically extract ingredients from any recipe into an interactive, persistent checklist.
- **Professional PDF Export:** Convert any recipe into a clean, printable PDF document directly from the app.
- **Social Sharing:** Share recipe details and links through WhatsApp, Email, or other social apps via Android Sharesheet.

## 📋 Requirements & Tech Stack
- **IDE:** Android Studio Ladybug (2024.2.1) or higher.
- **Language:** Java / XML.
- **JDK:** 17+
- **Minimum SDK:** API 24 (Android 7.0).
- **Backend:** Firebase (Firestore, Authentication, Storage).
- **Libraries:** Picasso/Glide (Image Loading), Android PdfDocument, Material Design Components.
